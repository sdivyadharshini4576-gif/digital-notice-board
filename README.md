# 📢 Digital Notice Board

A simple Digital Notice Board application that allows administrators to post, update, and manage notices digitally.
Users can view the latest announcements anytime, replacing traditional paper notice boards.

## Features
- Add, edit, and delete notices
- View latest announcements
- Admin login (if available)
- Simple and responsive interface

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP & MySQL (if applicable)
